6.1 Random walk with drift

A univariate random walk with drift observed with error is: \[\begin{equation} \begin{gathered} x_{t} = x_{t-1} + u + w_{t}, \text{ } w_t \sim \,\text{N}(0,q) \\ y_{t} = x_{t} + v_{t}, \text{ } v_t \sim \,\text{N}(0,r) \end{gathered} \tag{6.2} \end{equation}\]

Create some data from this equation: 

u <- 0.01
r <- 0.01
q <- 0.1
TT <- 100
yt <- cumsum(rnorm(TT, u, sqrt(q))) + rnorm(TT, 0, sqrt(r))

Fit with MARSS():

fit <- MARSS(yt)
